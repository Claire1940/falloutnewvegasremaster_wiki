<Section icon={Target} title="Пошаговое руководство по установке в Steam">

Чтобы ваша установка **Fallout New Vegas steam** работала безупречно с выбранным паком модов, необходимо следовать четкой последовательности действий. Начало с абсолютно «чистого листа» — самый важный фактор в предотвращении конфликтов порядка загрузки.

<Steps>
  <Step num="1" title="Чистая установка через Steam">
    Удалите все существующие версии игры через Steam. Перейдите в директорию Steam и вручную удалите все оставшиеся папки в common files. Переустановите игру в незащищенную папку (например, на другой диск). Запустите лаунчер один раз, чтобы создать конфигурационные файлы, затем закройте его.
  </Step>
  <Step num="2" title="Применение основных патчей движка">
    Скачайте New Vegas Script Extender (xNVSE) и 4GB Patcher. Запустите исполняемый файл 4GB Patcher прямо в папке с игрой. Эта модификация позволяет игре использовать вдвое больше памяти, что радикально снижает количество вылетов из-за нехватки ресурсов.
  </Step>
  <Step num="3" title="Настройка менеджера модов">
    Установите Mod Organizer 2 (рекомендуется для Viva New Vegas) или Vortex (рекомендуется для автоматизированных коллекций Nexus). Свяжите менеджер с папкой установки Steam. Убедитесь, что профиль чист и в виртуальной директории нет старых файлов.
  </Step>
  <Step num="4" title="Развертывание модов и исправление UI">
    Скачайте и разверните выбранный пак. Если вы используете коллекцию New New Vegas, запустите игру и сразу откройте меню паузы. Нажмите кнопку настроек интерфейса, чтобы отключить огромный трекер квестов, который иначе перекрывает значительную часть экрана.
  </Step>
</Steps>

В следующей таблице перечислены обязательные утилиты, необходимые для создания стабильного фундамента любой модифицированной сборки.

| Название инструмента | Основная функция | Важность |
| :--- | :--- | :--- |
| **xNVSE** | Расширяет возможности скриптов для сложных модов | **Обязательно** для 95% модов |

| **Vortex / MO2** | Управляет конфликтами файлов и виртуализирует папку | **Обязательно** для чистоты папки игры |
