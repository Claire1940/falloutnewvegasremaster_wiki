<Section icon={Target} title="Пошаговое руководство по ремастеру Fallout New Vegas">

Развертывание основных исправлений движка и расширителей скриптов требует точной последовательности установки. Выполните следующие шаги, чтобы создать высокостабильную среду с помощью Vortex Mod Manager.

<Steps>
  <Step num="1" title="Настройка менеджера модов">
    Скачайте и установите Vortex Mod Manager. Войдите в свой аккаунт Nexus Mods, перейдите на вкладку игр, найдите New Vegas и нажмите «Управлять» (manage). Перейдите в настройки (settings), интерфейс (interface) и включите управление профилями (profile management). Создайте новый профиль с именем «Modded» со своими настройками игры и файлами сохранений.
  </Step>
  <Step num="2" title="Инвалидация архивов">
    В Vortex перейдите в настройки и выберите вкладку «Workarounds». Установите флажок для «BSA Redirection» в разделе «Archive Invalidation». Это заставляет движок игры отдавать приоритет измененным отдельным файлам перед сжатыми игровыми архивами по умолчанию.
  </Step>
  <Step num="3" title="Стабильная коллекция">
    Найдите коллекцию «Stable New Vegas Collection» на Nexus Mods. Нажмите «Add collection», чтобы открыть Vortex. Выберите свой модифицированный профиль и начните загрузку. Эта коллекция автоматически устанавливает исправления ошибок, оптимизации движка и улучшения производительности.
  </Step>
  <Step num="4" title="Запуск патча 4 ГБ">
    Откройте папку с игрой через Vortex (Mods -> Open -> Open Game Folder). Найдите файл с именем `FNVpatch.exe` и дважды щелкните его, чтобы запустить. Окно командной строки подтвердит, что исполняемый файл был успешно пропатчен для использования большего объема системной памяти.
  </Step>
  <Step num="5" title="Исправление мастер-файлов игры">
    Запустите инструмент «Ultimate Edition ESM Fixes Remastered». Скопируйте адрес папки модов из Vortex, запустите инструмент установки, вставьте путь к каталогу и выполните установку. По завершении включите все исправленные файлы ESM в списке плагинов Vortex.
  </Step>
</Steps>

| Название утилиты | Основная функция | Метод установки |
| :--- | :--- | :--- |
| **4GB Patcher** | Расширяет адресное пространство виртуальной памяти | Ручной запуск исполняемого файла |
| **ESM Fixes** | Исправляет поврежденные ванильные мастер-файлы | Автоматическое развертывание инструмента |
| **BSA Redirection** | Заставляет движок загружать модифицированные ассеты | Переключатель в настройках Vortex |
| **NVSE** | Расширяет возможности скриптов для продвинутых модов | Автоматически через коллекции |

<Callout type="success" title="Совет по патчеру">
Запуск 4GB Patcher в версии GOG все равно рекомендуется. Хотя GOG предварительно патчит исполняемый файл для использования памяти, запуск патчера настраивает движок на автоматическую загрузку New Vegas Script Extender.

| Настройка | Рекомендуемое значение | Влияние на производительность |
| :--- | :--- | :--- |
