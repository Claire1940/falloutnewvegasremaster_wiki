<Section icon={Target} title="Schritt-für-Schritt-Anleitung zur Steam-Installation">

Um sicherzustellen, dass Ihre **Fallout New Vegas steam** Installation mit dem gewählten Mod-Paket einwandfrei läuft, müssen Sie eine strukturierte Installationssequenz befolgen. Ein völlig sauberer Neuanfang ist der kritischste Faktor, um Konflikte in der Ladereihenfolge zu vermeiden.

<Steps>
  <Step num="1" title="Saubere Steam-Installation durchführen">
    Deinstallieren Sie alle vorhandenen Versionen des Spiels über Steam. Navigieren Sie zu Ihrem Steam-Verzeichnis und löschen Sie manuell alle verbleibenden Ordner in den Common-Dateien. Installieren Sie das Spiel in einen nicht geschützten Ordner (z. B. auf einer zweiten Festplatte) neu. Starten Sie den Game-Launcher einmal, um die erforderlichen Konfigurationsdateien zu generieren, und schließen Sie ihn dann wieder.
  </Step>
  <Step num="2" title="Core-Engine-Patches anwenden">
    Laden Sie den New Vegas Script Extender (xNVSE) und den 4GB Patcher herunter. Führen Sie die ausführbare Datei des 4GB Patchers direkt in Ihrem Spielverzeichnis aus. Diese Modifikation ermöglicht es der ausführbaren Spieldatei, auf das Doppelte ihres Standard-Speicherlimits zuzugreifen, was Out-of-Memory-Abstürze drastisch reduziert.
  </Step>
  <Step num="3" title="Mod-Manager einrichten">
    Installieren Sie Mod Organizer 2 (empfohlen für Viva New Vegas) oder Vortex (empfohlen für automatisierte Nexus-Sammlungen). Verknüpfen Sie den Manager mit Ihrem Steam-Installationsverzeichnis. Stellen Sie sicher, dass Ihr Profil sauber ist und keine alten, losen Dateien im virtuellen Verzeichnis aktiv bleiben.
  </Step>
  <Step num="4" title="Mod-Pakete bereitstellen und UI fixen">
    Laden Sie Ihr gewähltes Paket herunter und stellen Sie es bereit. Wenn Sie die New New Vegas-Sammlung verwenden, starten Sie das Spiel und öffnen Sie sofort das Pausenmenü. Drücken Sie die entsprechende Taste für die UI-Optionen, um den übergroßen Quest-Tracker zu deaktivieren, der sonst einen erheblichen Teil Ihres Bildschirms verdeckt.
  </Step>
</Steps>

Die folgende Tabelle listet die obligatorischen Hilfswerkzeuge auf, die erforderlich sind, um ein stabiles Fundament für jedes gemoddete Setup zu schaffen.

| Tool-Name | Kernfunktion | Wichtigkeit |
| :--- | :--- | :--- |
| **xNVSE** | Erweitert die Scripting-Fähigkeiten für komplexe Mods | **Obligatorisch** für 95 % der modernen Mods |
| **4GB Patcher** | Erlaubt der Engine die Nutzung von 4 GB System-RAM | **Obligatorisch**, um Speicherabstürze zu verhindern |
| **New Vegas Tick Fix** | Behebt Physik-Bugs bei hohen Frameraten und Mikroruckler | Dringend empfohlen für Windows 10/11 |
| **Vortex / MO2** | Verwaltet Dateikonflikte und virtualisiert das Verzeichnis | **Obligatorisch**, um das Basisverzeichnis sauber zu halten |
