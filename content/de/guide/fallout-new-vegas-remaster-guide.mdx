Bevor Sie Mods herunterladen, müssen Sie Ihre Systemumgebung vorbereiten. Windows-Sicherheitsfunktionen schränken häufig Dateiänderungen in Standardverzeichnissen ein, was Script-Extender und benutzerdefinierte Assets blockieren kann.

| Anforderung | GOG-Version | Steam-Version | Microsoft Store |
| :--- | :--- | :--- | :--- |
| **Mod-Kompatibilität** | Hervorragend | Hervorragend | Schlecht (eingeschränkt) |
| **4GB Patch** | Vor-gepatcht | Erfordert manuellen Patch | Inkompatibel |
| **DLC-Anforderung** | Ultimate Edition | Ultimate Edition | Standard/Ultimate |
| **Installationspfad** | Benutzerdefinierter Ordner | Benutzerdefinierter Ordner | Standard Windows |

Um eine reibungslose Einrichtung zu gewährleisten, erstellen Sie einen dedizierten Ordner außerhalb des Programmverzeichnisses Ihres primären Systemlaufwerks (zum Beispiel `D:\Games\SteamLibrary`). Starten Sie nach der Installation den Game-Launcher mindestens einmal. Dieser erste Durchlauf ist entscheidend, da er Ihre Systemhardware automatisch erkennt, Ihre Display-Auflösung konfiguriert und die erforderlichen `.ini`-Konfigurationsdateien in Ihrem Dokumente-Ordner generiert.

<Callout type="warning" title="Plattform-Warnung">
Versuchen Sie nicht, die Microsoft Store-Version des Spiels zu modden. Sie sperrt Verzeichnisberechtigungen, was es unglaublich schwierig macht, wichtige Script-Extender und benutzerdefinierte Stabilitäts-Patches auszuführen.
</Callout>

<Section icon={Target} title="Schritt-für-Schritt Fallout New Vegas Remaster Guide">

Das Implementieren von Kern-Engine-Fixes und Script-Extendern erfordert eine präzise Abfolge von Installationen. Folgen Sie diesen Schritten, um mit dem Vortex Mod Manager eine hochstabile Umgebung aufzubauen.

<Steps>
  <Step num="1" title="Einrichtung des Mod-Managers">
    Laden Sie den Vortex Mod Manager herunter und installieren Sie ihn. Melden Sie sich bei Ihrem Nexus Mods-Konto an, navigieren Sie zum Tab "Games", suchen Sie nach New Vegas und klicken Sie auf "Manage". Gehen Sie zu den Einstellungen, Interface, und aktivieren Sie die Profilverwaltung. Erstellen Sie ein neues Profil namens "Modded" mit eigenen Spieleinstellungen und Speicherständen.
  </Step>
  <Step num="2" title="Archiv-Invalidierung">
    Navigieren Sie in Vortex zu den Einstellungen und wählen Sie den Tab "Workarounds". Aktivieren Sie das Kontrollkästchen für "BSA Redirection" unter "Archive Invalidation". Dies zwingt die Game-Engine, modifizierte lose Dateien gegenüber den standardmäßigen komprimierten Spielarchiven zu priorisieren.
  </Step>
  <Step num="3" title="Die stabile Sammlung">
    Suchen Sie die "Stable New Vegas Collection" auf Nexus Mods. Klicken Sie auf "Add Collection", um Vortex zu öffnen. Wählen Sie Ihr gemoddetes Profil aus und starten Sie den Download. Diese Sammlung installiert automatisch Fehlerbehebungen, Engine-Optimierungen und Leistungsverbesserungen.
  </Step>
  <Step num="4" title="4GB-Patch ausführen">
    Öffnen Sie Ihren Spielordner über Vortex (Mods -> Open -> Open Game Folder). Suchen Sie die Datei `FNVpatch.exe` und führen Sie sie per Doppelklick aus. Ein Befehlsfenster bestätigt, dass die ausführbare Datei erfolgreich gepatcht wurde, um mehr Systemspeicher zu nutzen.
  </Step>
  <Step num="5" title="Spiel-Masterdateien reparieren">
    Führen Sie das Tool "Ultimate Edition ESM Fixes Remastered" aus. Kopieren Sie die Adresse des Mod-Ordners aus Vortex, führen Sie das Installer-Tool aus, fügen Sie den Verzeichnispfad ein und führen Sie die Installation aus. Aktivieren Sie nach Abschluss alle reparierten ESM-Dateien in Ihrer Vortex-Plugin-Liste.
  </Step>
</Steps>

| Name des Dienstprogramms | Hauptfunktion | Installationsmethode |
| :--- | :--- | :--- |
| **4GB Patcher** | Erweitert den virtuellen Speicheradressraum | Manueller Start der EXE |
| **ESM Fixes** | Repariert beschädigte Vanilla-Masterdateien | Automatisierte Tool-Bereitstellung |
| **BSA Redirection** | Erzwingt das Laden modifizierter Assets | Vortex-Workaround-Umschalter |
| **NVSE** | Erweitert Scripting-Fähigkeiten für fortgeschrittene Mods | Automatisiert über Sammlungen |

<Callout type="success" title="Patcher-Tipp">
Die Ausführung des 4GB Patchers auf der GOG-Version wird weiterhin empfohlen. Obwohl GOG die EXE-Datei für die Speichernutzung vor-patcht, konfiguriert die Ausführung des Patchers die Engine so, dass der New Vegas Script Extender automatisch geladen wird.
</Callout>
