<Section icon={Target} title="Step-by-Step Fallout New Vegas Remaster Guide">

Deploying core engine fixes and script extenders requires a precise sequence of installations. Follow these steps to build a highly stable environment using Vortex Mod Manager.

<Steps>
  <Step num="1" title="Mod Manager Setup">
    Download and install Vortex Mod Manager. Log in to your Nexus Mods account, navigate to the games tab, search for New Vegas, and click manage. Go to settings, interface, and enable profile management. Create a new profile named "Modded" with its own game settings and save files.
  </Step>
  <Step num="2" title="Archive Invalidation">
    In Vortex, navigate to settings and select the Workarounds tab. Check the box for BSA Redirection under Archive Invalidation. This forces the game engine to prioritize modified loose files over the default compressed game archives.
  </Step>
  <Step num="3" title="The Stable Collection">
    Locate the Stable New Vegas Collection on Nexus Mods. Click add collection to open Vortex. Select your modded profile and begin the download. This collection automatically deploys bug fixes, engine optimizations, and performance enhancers.
  </Step>
  <Step num="4" title="Execute the 4GB Patch">
    Open your game folder via Vortex (Mods -> Open -> Open Game Folder). Locate the file named `FNVpatch.exe` and double-click to run it. A command window will confirm that the executable has been successfully patched to utilize more system memory.
  </Step>
  <Step num="5" title="Fix Game Master Files">
    Run the Ultimate Edition ESM Fixes Remastered tool. Copy the mod folder address from Vortex, run the installer tool, paste the directory path, and execute the installation. Once finished, enable all fixed ESM files in your Vortex plugins list.
  </Step>
</Steps>

| Utility Name | Primary Function | Installation Method |
| :--- | :--- | :--- |
| **4GB Patcher** | Expands virtual memory address space | Manual executable run |
| **ESM Fixes** | Repairs corrupted vanilla master files | Automated tool deployment |
| **BSA Redirection** | Forces the engine to load modified assets | Vortex workaround toggle |
| **NVSE** | Expands scripting capabilities for advanced mods | Automated via Collections |
