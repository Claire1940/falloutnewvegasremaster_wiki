<Section icon={Target} title="Step-by-Step Steam Installation Guide">

To ensure your **Fallout New Vegas steam** installation runs flawlessly with your chosen mod pack, you must follow a structured installation sequence. Starting from a completely clean slate is the most critical factor in avoiding load-order conflicts.

<Steps>
  <Step num="1" title="Perform a Clean Steam Installation">
    Uninstall any existing versions of the game through Steam. Navigate to your Steam directory and manually delete any leftover folders in the common files. Reinstall the game into a non-protected folder (such as a secondary drive). Run the game launcher once to generate the necessary configuration files, then close it.
  </Step>
  <Step num="2" title="Apply Core Engine Patches">
    Download the New Vegas Script Extender (xNVSE) and the 4GB Patcher. Run the 4GB Patcher executable directly inside your game directory. This modification allows the game executable to access double its default memory limit, which drastically reduces out-of-memory crashes.
  </Step>
  <Step num="3" title="Set Up Your Mod Manager">
    Install Mod Organizer 2 (recommended for Viva New Vegas) or Vortex (recommended for automated Nexus collections). Link the manager to your Steam installation directory. Ensure your profile is clean and that no old, loose files remain active in the virtual directory.
  </Step>
  <Step num="4" title="Deploy Mod Packs and Fix the UI">
    Download and deploy your chosen pack. If you are using the New New Vegas collection, launch the game and immediately open the pause menu. Press the designated UI options button to disable the oversized quest tracker, which otherwise covers a significant portion of your screen.
  </Step>
</Steps>
