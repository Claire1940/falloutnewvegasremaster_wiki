<Section icon={Shield} title="The Pillars of Stability: NVSE and 4GB Patcher">

The most important step in learning **Fallout New Vegas how to mod** is installing the New Vegas Script Extender (NVSE). This tool expands the game's scripting capabilities, allowing complex mods to function. Without it, most modern mods simply will not load. Following NVSE, the 4GB Patcher is non-negotiable; it allows the game to utilize 4GB of RAM instead of the default 2GB, drastically reducing "Out of Memory" crashes.

| Tool Name | Primary Function | Installation Method |
| :--- | :--- | :--- |
| **xNVSE** | Modern branch of the script extender with bug fixes | Manual drag-and-drop to root folder |
| **4GB Patcher** | Upgrades the .exe to be Large Address Aware | Run the .exe inside the game folder |
| **HEAPS** | Optimized memory allocation for the game engine | Plugin for NVSE |
| **NVTF** | New Vegas Tick Fix for smooth high-framerate play | Mod Manager installation |

<Callout type="tip" title="Pro Tip: Verification">
Once you have installed NVSE, launch the game, open the console with the `~` key, and type `GetNVSEVersion`. If the console returns a version number, you have successfully completed the most difficult part of the setup.
</Callout>

<Section icon={Zap} title="Step-by-Step Installation Process">

Follow these steps to ensure your environment is ready for advanced gameplay modifications. This sequence is designed to prevent the most common errors encountered by new modders.

<Steps>
  <Step num="1" title="Initial Game Cleanup">
    Uninstall the game and delete the "Fallout New Vegas" folder in SteamApps/common. Reinstall to ensure a 100% clean slate. Launch the game once to the main menu and then exit.
  </Step>
  <Step num="2" title="Manual Tool Deployment">
    Download xNVSE and the 4GB Patcher. Extract the xNVSE files into the folder containing FalloutNV.exe. Move the Patcher .exe into the same folder and run it as an administrator.
  </Step>
  <Step num="3" title="Mod Manager Configuration">
    Install Vortex or MO2. Link your Nexus Mods account in the settings. Direct the manager to your Fallout New Vegas installation path if it is not detected automatically.
  </Step>
  <Step num="4" title="Essential Bug Fixing">
    Download and install "Yukichigai Unofficial Patch (YUP)" via your mod manager. This mod fixes thousands of bugs left by the original developers without changing gameplay balance.
  </Step>
  <Step num="5" title="Deployment and Testing">
    Click "Deploy" in your manager. Launch the game using the "New Vegas" shortcut (the 4GB patcher makes it so you don't need a special NVSE shortcut anymore) and verify stability.
  </Step>
</Steps>
