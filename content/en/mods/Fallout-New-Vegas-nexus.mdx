<Section icon={Sparkles} title="Essential Stability and Scripting Tools">

To run modern mods from the **Fallout New Vegas nexus**, you must bypass the limitations of the original 2010 game engine. The base game is restricted to 2GB of RAM and uses an outdated scripting language that cannot handle complex AI or high-resolution textures.

**The Holy Trinity of Stability:**
1.  **NVSE (New Vegas Script Extender)**: This tool expands the game's scripting capabilities. Without it, mods that add new UI elements, complex quests, or dynamic weather simply will not load.
2.  **4GB Patcher**: This executable modifies the game to recognize 4GB of RAM instead of 2GB. This is the single most effective way to prevent the "Out of Memory" crashes common in the Mojave.
3.  **NVAC (New Vegas Anti-Crash)**: A specialized plugin that catches common engine errors before they cause a desktop crash, allowing the game to continue running through minor glitches.

| Tool Name | Primary Function | Installation Method |
| :--- | :--- | :--- |
| **NVSE** | Script Expansion | Manual (Root Folder) |
| **4GB Patcher** | Memory Expansion | Run Executable Once |
| **NVAC** | Crash Prevention | Install via Vortex |
| **Stewie Tweaks** | Engine Bug Fixes | Install via Vortex |

<Callout type="warning" title="Critical Warning">
The 4GB Patcher must be run from your game's root directory (where `FalloutNV.exe` is located). If you run it from your Downloads folder, it will not find the game and will fail to apply the patch.
</Callout>

</Section>

<Section icon={Target} title="Step-by-Step Mod Installation Guide">

Once your stability tools are in place, you can begin exploring the creative side of the **Fallout New Vegas nexus**. Following a specific order ensures that textures, meshes, and scripts interact correctly without overwriting vital data.

<Steps>
  <Step num="1" title="Link Vortex to Nexus">
    Download and install Vortex. Log in with your Nexus account and go to the 'Games' tab. Search for 'Fallout: New Vegas' and click 'Manage'. Vortex will automatically detect your installation path.
  </Step>
  <Step num="2" title="Apply the 4GB Patch">
    Download the 4GB Patcher from Nexus. Extract the `.exe` file into your game folder. Double-click it; a command window will appear and confirm 'Game Patched'. You can now delete the patcher `.exe`.
  </Step>
  <Step num="3" title="Install NVSE">
    While Vortex can sometimes automate this, manual installation is safer. Drag the `.dll` and `.exe` files from the NVSE archive into your game's root directory.
  </Step>
  <Step num="4" title="Download Your First Mod">
    Browse the Nexus website. Find a mod like 'The Strip Open' or 'Sprint Mod'. Click the 'Vortex' button on the mod page. Vortex will handle the download and prompt you to 'Install' and 'Enable'.
  </Step>
  <Step num="5" title="Deploy and Play">
    Click the 'Deploy Mods' button in Vortex. This finalizes the file links. Launch the game using the `nvse_loader.exe` or the play button inside Vortex to ensure all scripts are active.
  </Step>
</Steps>

<Callout type="success" title="Verification">
To verify NVSE is working, open the in-game console (tilde key `~`) and type `GetNVSEVersion`. If it returns a number, your installation was successful.
</Callout>

</Section>
