<Section icon={Wrench} title="Installation: The 2026 Step-by-Step Process">

Installing a **fallout new vegas remaster mod list** has become significantly easier thanks to "Collections." Instead of downloading 100+ mods individually, you can now use automated tools to handle the heavy lifting.

<Steps>
  <Step num="1" title="Clean Installation">
    Start with a fresh install of Fallout: New Vegas. If you previously modded the game, delete the entire folder in SteamApps and the configuration files in 'My Documents'.
  </Step>
  <Step num="2" title="Mod Manager Setup">
    Download and install **Vortex** or **Mod Organizer 2**. Vortex is recommended for beginners using Nexus Collections, while MO2 offers more control for advanced users.
  </Step>
  <Step num="3" title="Apply the 4GB Patcher">
    Run the 4GB Patcher tool. Verify it worked by checking the 'nvse_loader.log' or using a console command in-game to check memory allocation.
  </Step>
  <Step num="4" title="Install the Stability Foundation">
    Search for the **Stability Collection** on Nexus Mods. This should be the first collection you install to ensure the engine is bug-free before adding graphics.
  </Step>
  <Step num="5" title="Layer the Remaster Collections">
    Add the **Visual Remaster** and **UI Collections**. Ensure you follow the "Load Order" instructions provided by the collection author to prevent overwriting critical files.
  </Step>
</Steps>

<Callout type="warning" title="Load Order Matters">
Always use **LOOT (Load Order Optimisation Tool)** or the built-in sorter in Vortex. An incorrect load order is the most common cause of "missing mesh" errors (the dreaded red exclamation marks).
</Callout>

</Section>
