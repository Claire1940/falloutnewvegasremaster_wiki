| File Name | Location | Attribute Change |
| :--- | :--- | :--- |

| **falloutprefs.ini** | Documents/My Games/FalloutNV | Set to Read-Only |
| **fallout_default.ini** | Game Root Directory | Set to Read-Only |

<Callout type="warning" title="Important Warning">

</Callout>

By setting these files to read-only, you ensure that the game engine respects your custom modifications and performance tweaks, regardless of how you launch the application.
