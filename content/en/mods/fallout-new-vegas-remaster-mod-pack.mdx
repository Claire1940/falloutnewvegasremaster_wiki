<Section icon={Target} title="Step-by-Step Installation Guide">

Installing a massive selection of mods manually is highly prone to user error. Modern modding utilizes automated installers to handle the heavy lifting, ensuring proper load orders and conflict resolution. Following a structured installation process is critical to achieving a stable, crash-free game.

<Steps>
  <Step num="1" title="Clean Game Installation">
    Ensure you have a fresh install of Fallout New Vegas: Ultimate Edition via Steam or GOG. Run the game launcher once to generate the default configuration files.
  </Step>
  <Step num="2" title="Apply Engine Patches">
    Download and run the 4GB Patcher utility. This allows the game executable to utilize 4GB of virtual memory instead of the default 2GB, preventing frequent crashes.
  </Step>
  <Step num="3" title="Configure Mod Manager">
    Set up Vortex or Mod Organizer 2. Ensure Archive Invalidation is enabled under your mod manager settings to allow custom textures to load.
  </Step>
  <Step num="4" title="Run the Automated Installer">
    Use WabbitJack or Nexus Collections to download your selected mod pack. Follow the on-screen prompts to automatically download and install all required files.
  </Step>
</Steps>

| Error / Symptom | Probable Cause | Recommended Solution |
|---|---|---|
| Crash on Startup | Missing master files or unpatched executable | Run 4GB Patcher and verify load order in mod manager |
| Missing Textures | Archive Invalidation disabled | Toggle Archive Invalidation off and on in Vortex/MO2 |
| Severe Stuttering | High-resolution texture overhead | Install performance-friendly LOD files or reduce texture resolution |
| Infinite Loading Screen | Save game corruption or engine limit | Install the Infinite Loading Screen Fix and Tick Fix mods |

<Callout type="info" title="Installation Tip">
